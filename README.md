# Система учёта заявок на техническое обслуживание

## Назначение
Программный модуль для учёта заявок на техническое обслуживание в организации.

## Основные функции
- Создание новых заявок
- Просмотр всех заявок
- Изменение статуса заявок
- Поиск заявок по различным критериям
- Сохранение данных

## Как запустить
1. Убедитесь, что установлен .NET SDK 5.0 или выше
2. Выполните команду: `dotnet run`
